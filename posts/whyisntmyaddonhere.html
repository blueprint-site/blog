How to see your addon on Blueprint

There are a couple of things you need to do.
    1. First: publish it on Modrinth. The Curseforge support will be added soon, presumably when API is completed and when we make the V2.

    2. It has to have "create" either in it's title (name) or in it's description (under the title). If not, Labrinth (Modrinth API) will not find it, so we won't know it either.
    2a. What if my addon doesn't have "create" in title/description and i don't want to add it? We can (hopefully) add it manually to our site.
    Note: The manual addition feature is coming soon with the API and with the V2.
    2b. How to manually add it? Message us on discord: link to discord

    3. That's all! The addon should be on the page if you follow these 2 simple steps.
